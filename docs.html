Quickstart Guide

Welcome to the ATHENA documentation. In this guide, you'll learn how to list your first project and connect with other students.

1. Create your account

Sign up with your student email to access platform features. Navigate to your Developer Settings dashboard to complete your developer profile.

2. Submit a Project via API

curl -X POST https://api.studenthub.com/v1/projects \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "title": "My Awesome App",
  "repo_url": "https://github.com/user/repo",
  "tags": ["react", "nodejs", "mongodb"],
  "description": "<p>A full-stack portfolio builder.</p>"
}'

If successful, you will receive a 200 OK response and your project will be live on the dashboard!
